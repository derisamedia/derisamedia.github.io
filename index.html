How To Install Theme in another Firmware non Alpha OS

If you have a problem to install it

1
Download The Ipk

On the tab releases in the luci-theme-alpha repositories

2
open Your Luci Openwrt dasboard

and click tabsystem>software.

3
Click Upload Package

Upload Ipk and install and set in the system>system>language and style, Select Alpha in design column and save And Apply.
